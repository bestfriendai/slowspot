## Google Play Store Localization

Google Play uses similar metadata structure. Copy the same content with these field name mappings:

| App Store Connect | Google Play Console |
|-------------------|---------------------|
| App Name | Title |
| Subtitle | Short Description |
| Description | Full Description |
| Keywords | N/A (Google extracts from description) |
| Promotional Text | N/A |

**Google Play Additional Requirements:**
- Short Description: 80 chars max (use Subtitle)
- Full Description: 4000 chars max (use Description)
- Feature Graphic: 1024x500 PNG/JPG
- Screenshots: Min 2, Max 8 per device type

## App Store Connect Setup Instructions

1. Go to App Store Connect > My Apps > Slow Spot
2. Select the App Information section
3. For each language, click "+" to add localization
4. Copy the appropriate content from above
5. Save changes
